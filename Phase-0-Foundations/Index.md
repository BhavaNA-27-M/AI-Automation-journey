# Phase 0 — Foundations of AI & Automation

This is the **thinking and mindset phase** of my AI Automation Mastery roadmap.  
No tools yet — just clarity, logic, and system design.

---

## 🔹 Topics Index

1. **0.1 — AI vs ML vs Automation**  
   - Understand the difference between intelligence, learning, and rules.

2. **0.2 — Workflow Concept**  
   - Core of every system: Input → Process → Output.

3. **0.3 — Input / Process / Output Deep Dive**  
   - Types of input, process, and output engineers use.

4. **0.4 — API Fundamentals**  
   - How systems communicate and exchange data.

5. **0.5 — If This → Then That Logic**  
   - Decision-making rules that power automation.

6. **0.6 — Types of Data**  
   - Structured vs unstructured, and when AI is needed.

7. **0.7 — Basic Logic Thinking**  
   - Mini programming mindset: goal, input, process, output.

8. **0.8 — System Thinking**  
   - Design systems, not just tools.

9. **0.9 — Types of Automation**  
   - Task Automation, Workflow Automation, Intelligent Automation, Autonomous Systems.

10. **0.10 — Errors & Edge Cases Thinking**  
    - Prepare for real-world failures and exceptions.

---

## 📌 Goal
By the end of Phase 0, **think like an AI Automation Engineer** — ready to design workflows, understand data, and handle edge cases before using any tools.
